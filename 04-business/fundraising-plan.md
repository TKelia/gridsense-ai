## Principle: you need almost no money to WIN the capstone
The capstone deliverable (software demo + feasibility pack + elite pitch) costs ~**$0** (free cloud tiers, your laptop/Wi-Fi, open-source designs, ALU resources). A full physical kit is only **~RWF 69,100 (~$47)** and a bench PoC **~RWF 40,100 (~$27)** if you want the bonus tier (`04-business/capex-opex.md`). **Funding is for scaling, not for passing.**

## Tier 0 — Free / in-kind (do FIRST, this week)

3. **Borrow one kit's parts** — a single ESP32 + one SCT-013 (~RWF 27,500 retail) is enough for a bench PoC.

## The lean ladder (how the money compounds)
```
$0 capstone demo  →  enter Hanga / FEC / Youth4Climate  →  win prize/grant
   →  build 3–5 real kits + 1 pilot home/landlord  →  pilot data + testimonial
   →  larger grant/accelerator (EEP, ClimaFii, Katapult)  →  small production run
```
Each rung needs only the proof from the rung below — never a big upfront ask.
